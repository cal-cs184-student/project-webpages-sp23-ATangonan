PowerWash Simulator

Summary

We are going to simulate fast-moving pressurized water through different nozzles with fluid simulations. This will be an interactive simulation game where the objective is to clean a certain object or area.

Problem Description

Describe the problem that you are trying to solve, why it is important, where it is challenging. Give us a general idea on how you are going to solve it.

Problem(s) to be solved:

  • Create at least 2 assets for clean/dirty objects and blend them or morph them
  • How to model the objects such that only a specific area of it can get washed at a time
  • How would the tool spray water? How would the water interact with the object?

Importance

  • Without the assets, we would have nothing to clean or use to wash. There needs to be accuracy with what is being washed at a certain time in order to ensure the simulation works accordingly. Water interacting with the dirty areas is once of the main mechanics of the game.

Challenge(s)

  • How to integrate Unity. Ensure no one messes up the Git repo for the project. Having a good efficient algorithm and run time.

Plan of Attack

  • Create a game world. Plan out the physics behind each tool. Work on how the water will interact with objects and the environment. Set some kind of end goal for the player or incentive them to keep playing.
  • The player will be able to walk around an environment and use a variety of tools with differing custom physics to manipulate the environment.
  • We can measure fps using some kind of built in Unity system
